# Retro Apple-Inspired UI Guidance

Use these rules for Apple-like itinerary sites with a warm Tainan retro palette. Do not copy Apple assets or brand marks.

- Palette: `#f5f1e9` page background, `#fffaf1` surfaces, `#251f1a` primary text, `#6f665e` secondary text, `#b94f32` action terracotta, `#1e6f68` route green, `#2f5f89` info blue, `#d79a2b` status gold.
- Typography: system UI stack, heavy hero headline, compact section headings, no negative letter spacing.
- Layout: large quiet hero, warm paper-like metric surfaces, sticky translucent topbar, generous whitespace, thin separators.
- Cards: maximum 8px radius, subtle shadow, no nested card frames.
- Actions: terracotta primary Google Map buttons; avoid loud gradients.
- Motion/feel: prefer calm clarity over decorative effects. Use blur only for topbar or small panels.
- Content: write itinerary facts and route reasoning; avoid visible text explaining website features.
- Mobile: stack cards, keep buttons full-width only under narrow screens, ensure no horizontal overflow.
